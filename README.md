# BudgetTracker

A full-stack personal finance tracking web application. Created to brush up on old app development skills but learn some new tools (Next.JS, TypeScript mostly)

## Overview

The goal of this project is to build a simple but useful finance tracker where users can:

- Track income and expenses
- Categorize transactions
- View basic spending summaries
- Visualize financial data over time

More features will be added as development continues.

---

## Tech Stack

### Frontend
- Next.js (React)
- TypeScript
- Tailwind CSS

### Backend
- FastAPI (Python)
- SQLAlchemy ORM

### Database
- PostgreSQL

### DevOps / Infrastructure
- Docker
- Docker Compose
- Planned deployment: AWS or Azure

---

## Planned Features

- User authentication (login/register)
- Add, edit, and delete transactions
- Transaction categories (food, rent, etc.)
- Monthly spending summaries
- Basic dashboard with charts
- Responsive UI for desktop and mobile

Currently in early Minimum Viable Product phase.
